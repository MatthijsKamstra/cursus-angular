# 01 Start

## Generate initial project

```bash
ng new 01_start
```

Kies voorlopig voor

- routing (y/n) - no
- css (select) - css

## Start

html and ts file

- `01_start/src/app/app.component.html`

## TL;DR

https://angular.io/guide/setup-local

```bash
ng new 01_start
cd 01_start
ng serve --open
```

will open your browser: [http://localhost:4200/](http://localhost:4200/)
